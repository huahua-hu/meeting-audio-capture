# MeetingAudioCapture

A local-only macOS menu-bar app that records system audio and microphone input as separate tracks for reviewing meetings, interviews, and conversations.

## Status

The project requires macOS 15 or later and Xcode 16 or later. The first release is source-only and does not include a signed or notarized installer.

> Follow applicable laws, meeting rules, and organizational policies. Obtain consent from participants when required.
